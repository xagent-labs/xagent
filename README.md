# XAgent — AI Agent Builder, Runtime, Marketplace & Agentic Payments

**XAgent** (also written **X-Agent**) is an AI agent builder, runtime, and marketplace that turns an idea described in plain language into a working AI agent or AI‑powered web app — and lets you monetize it with built‑in **agentic payments**.

## What is XAgent?

XAgent (X-Agent) lets you **speak to build** an AI agent and **share to connect** it to the tools you already use. Describe the outcome in plain language; XAgent decides the steps, picks the tools, and ships a working agent or AI app — no brittle workflows, no boilerplate.

- 🛠️ **Agent Builder** — plain‑language prompt → a working AI agent or AI‑powered web app.
- ⚙️ **Runtime** — runs agent actions, tool calls, and multi‑agent clusters safely.
- 🏪 **Marketplace** — discover, share, and reuse agents and plugins.
- 💸 **Agentic Payments** — give agents budgets, approvals, and spend limits, and settle in stablecoins via [**xpense**](https://github.com/xagent-labs/xpense) and the `x402` protocol.
